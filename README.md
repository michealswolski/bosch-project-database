# Cross-Divisional Project Database
> Power Apps Canvas Application — Built During Automotive Supplier Internship

## Overview
Enterprise Canvas App system featuring dark futuristic UI themes, multi-step wizard workflows, admin management dashboards, and an integrated ticketing system. Deployed cross-tenant across global cybersecurity divisions.

## What I Built
- 11-screen Canvas App with dark futuristic theme design
- Multi-step wizard workflow for cybersecurity project intake
- Role-based admin dashboard with ticketing system
- Hardened `.msapp` deployment packages for cross-tenant compatibility
- Power Automate flows for automated notifications and approvals
- SharePoint list backend with relational data architecture

## Tech Stack
| Tool | Purpose |
|------|---------|
| Power Apps (Canvas) | Frontend application |
| Power Automate | Workflow automation and notifications |
| SharePoint Online | Backend data storage |
| PAC CLI | Application lifecycle management and packaging |

## Challenges Solved
- Troubleshot PAC CLI `.msapp` repack/import errors across different tenant environments
- Built deployment packages that work cross-tenant without breaking references
- Designed UI that matches modern SOC dashboard aesthetics within Power Apps constraints

## What I Learned
- Enterprise Power Apps ALM (Application Lifecycle Management)
- Cross-tenant deployment troubleshooting
- Canvas App performance optimization for complex forms
